# Vim - My concrete guide

## Versions of Vim

- vi
- vim
- Neovim
- Neovide

## Install Vim

### On Arch Linux

```bash
sudo pacman -S vim
```

### On Debian/Ubuntu

```bash
sudo apt install vim
```

## Install Neovim

### On Arch

```bash
sudo pacman -S neovim
```

## How to open Vim

```bash
> vim
```

## How to open neovim

```bash
> nvim
```

## How to exit vim

```bash
:q
```

## How to save and exit

```bash
:wq
```

## How to save and exit without saving

```bash
:w
```

## How to exit without saving

```bash
:q!
```
